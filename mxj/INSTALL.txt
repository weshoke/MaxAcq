MaxAcq is a Java (mxj) objet for Max (www.cycling74.com) for communicating with Biopac's Acq server.

Prerequisites: 
	- AcqKnowledge software with network capability from Biopac
	- Max5 from Cycling '74
	- Java runtime environment

Installation Instructions
	- copy all files in the 'classes' folder to:
		/Applications/Max5/Cycling '74/java/classes

	- copy all files in the 'lib' folder to:
		/Applications/Max5/Cycling '74/java/lib

	- copy all files in the 'help' folder to:
		/Applications/Max5/Cycling '74/java/help
